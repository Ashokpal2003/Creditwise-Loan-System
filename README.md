Loan Approval Prediction using Machine Learning

Project Overview

This project builds an end-to-end supervised Machine Learning pipeline to predict whether a loan application will be approved or rejected.
The model uses Logistic Regression and Naive Bayes algorithms to perform binary classification on loan data.

The project also includes Exploratory Data Analysis (EDA), feature engineering, model training, and model evaluation.
Finally, the model is deployed as a web application using Streamlit.

Objectives

- Predict loan approval status using machine learning.
- Perform data analysis to understand loan applicant patterns.
- Build and evaluate classification models.
- Deploy the model as an interactive web application.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

Machine Learning Models

The following algorithms are used:

- Logistic Regression
- Naive Bayes

These models are trained to classify whether a loan should be approved or rejected.

Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Deployment using Streamlit

Model Evaluation Metrics

The models are evaluated using:

- Precision
- Recall
- F1 Score

These metrics help measure the accuracy and performance of the classification model.

Deployment

The trained model is deployed using Streamlit, allowing users to enter applicant details and get a prediction for loan approval.
